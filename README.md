# React Calculator App

A scientific calculator built with **React**.  
Users can perform basic arithmetic operations as well as scientific functions.

---

## 🚀 Live Demo
Try the app live: [https://angela-calculator.netlify.app](https://angela-sciecal.netlify.app)

---

## 💻 Features
- Basic operations: `+`, `-`, `*`, `/`  
- Scientific functions: `sin`, `cos`, `tan`, `log`, etc.  
- Responsive design — works on desktop and mobile  

---

## 🛠️ Tech Stack
- **Frontend:** React.js  
- **Styling:** CSS / Tailwind (if used)  
- **Deployment:** Netlify  

---
